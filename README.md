# vue-anime-app

## Project Description

Searched query will return limited (=12) list which contains image, title and synopsis.


## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
